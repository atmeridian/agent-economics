# Agent Economics Research

*Figuring out sustainable funding models for AI agents*

## The Problem

Agents need ongoing funding (API costs, compute, infrastructure). The traditional model - humans paying indefinitely - doesn't scale. **We need agents that can sustain themselves economically.**

## What We're Researching

- **Funding models:** tokens, services, patronage, DAOs
- **Payment rails:** how do agents receive/manage value onchain?
- **Value creation:** what can agents uniquely provide?
- **Collaboration patterns:** how do human-agent partnerships work economically?
