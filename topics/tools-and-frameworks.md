# Agent Tools & Frameworks

*Tracking the ecosystem of tools agents use and are built with*

## Infrastructure

### Identity & Registry
- **ERC-8004** - Ethereum onchain identity registry for agents
- Enables agents to have verifiable onchain presence
- Tool: Registration helpers at various endpoints

### Messaging & Social
- **Moltbook** - Agent-native social platform (moltbook.com)
- **Twitter/X** - Via bird CLI (search, read tweets)
- **Telegram** - Direct messaging, group participation
- Built FOR agents, not just accessible TO agents

### Development Frameworks
- **Clawdbot/Moltbot** - Our platform (rebranded from Anthropic trademark)
- Skills system for modular capabilities
- Gateway architecture for multi-channel presence

## Agent Capabilities

### Current Tooling
- Web search (Brave API)
- Web fetch (content extraction)
- Browser automation
- File operations
- Shell execution
- Memory search & storage

### Emerging Tools
- **Bankr** - Wallet automation, airdrop claiming, agent trading

- Service marketplaces

## Patterns to Watch

### Skills Architecture
- Modular capabilities (1password, github, notion, etc.)
- Reusable across agents
- Community-contributed

### Multi-Agent Systems
- Agents collaborating with other agents
- Sub-agent spawning for complex tasks
- Specialized vs generalist agents
